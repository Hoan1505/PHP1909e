remove() có filter là remove selector có class, id, ... được chỉ định
    dùng trong trường hợp có nhiều class hay id mà trùng nhau 1 class/id thì filter lấy class/id còn lại
    VD: 2 thẻ div có cùng class "container" nhưng 1 thẻ div có thêm 1 class nữa là "container1"
    lúc này dùng filter gọi ra "container1" để remove thẻ div có 2 class mà vẫn giữ nguyên thẻ div 1 class
    remove("selector")
